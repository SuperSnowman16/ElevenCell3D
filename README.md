# ElevenCell3D

Bindings List:
- F1-F5 used to do 1-5 move scrambles
- F6 for a full scramble
- F12 for a reset
- Left / Right click to rotate a cell counterclockwise / clockwise around the ridge / edge / corner clicked
- Ctrl + click will do a full puzzle rotation around the same axis as the twists above
- Scroll wheel to zoom
- Left click drag to rotate
- Right click drag to do hyperbolic translations
- Middle click moves the cell clicked to the middle (similar to cell recentering in HSC)
- Ctrl + Middle click moves the piece to the origin (doesn't work for wings / petals)
- Ctrl+Z for Undo, Ctrl+Y or Ctrl+Shift+Z for Redo
- Ctrl+S to save a solve, Ctrl+O to open a solve
